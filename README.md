Códigos em Python voltados para Data Science, com meus exemplares de análises e de situações de negócio.
Todas as bases de dados são fictícias e/ou estão disponíveis na Internet.

## Bibliotecas/Módulos Utilizados
- `pandas`
- `sqlite3`
- `plotly.express`
- `seaborn`
- `statsmodels`


## Estrutura
- `notebooks/` - Arquivos .ipynb (Notebooks) com informações, comentários e o código;
- `data-scripts/` - Bases utilizadas e arquivos .py (código presente nos notebooks).
